数据结构

ECMAScript有5种简单数据结构（也称为基本数据结构类型）：Undefined、Null、Boolean、Number和String。还有一种复杂数据结构——Object。

typeof操作符

typeof用于检测给定变量的数据类型。以下表示对某个值使用的结果：

示例：


var message = 'some string';alert(typeof message); // "string"alert(typeof(message); // "string"alert(typeof 7); // "number"

Undefined类型

Undefined类型只有一个值undefined，未声明或未初始化的变量都使用typeof检测都会返回undefined。未初始化的变量的值就是undefined。

示例：


var first;alert(first); // undefinedalert(first == 'undefined'); // falsealert(first == undefined); // truealert(typeof(first) == undefined); // falsealert(typeof(first) == 'undefined'); // true
alert(typeof(second) == undefined); // false alert(typeof(second) == 'undefined') // truealert(second == 'undefined'); // error

Null类型

Null类型是第二个只有一个值的数据类型，这个特殊的值是null。null值表示一个空对象指针，所以使用typeof操作符检测null值时会返回“object”。

示例：


var first = null;alert(typeof first); // "object"
if (first != null) { // 一般使用情况    // 处理代码}


Boolean类型

Boolean类型有两个变量值：true和false。使用Boolean()函数可以将其他类型转换成boolean类型。

示例：


var first = 'Hello world!';var firstAsBoolean = Boolean(first);alert(firstAsBoolean); // true

Number类型

Number类型使用IEEE754格式来表示整数和浮点数值。

三种进制：十进制、八进制、十六进制。八进制字面值的第一位必须是零（0），然后是八进制数字序列数字序列（0~7）,如果字面量中的数值超出了范围，那么前导零将被忽略，后面的数值将被当做十进制数值解析。十六进制字面值的前两位必须是0x，后跟任何十六进制数字（0~9及A~F），字母A~F可以大写，也可以小写。


var intNum = 55; // 整数var octalNum1 = 070; // 八进制的56var octalNum2 = 079; // 无效的八进制数值——解析为79var octalNum3 = 08; // 无效八进制数值——解析为8
var hexNum1 = 0xA; // 十六进制的10var hexNum2 = 0x1f; // 十六进制的31


浮点数值：



var floatNum1 = 0.1;var floatNum2 = .1; // 有效，但不推荐var floatNum3 = 1.; // 1


e表示法：



var floatNum = 3.125e7; // 等于31250000(3.125乘以10的7次方）


提示（不要做这样的测试）：



if (a+b ==0.3) { // dosome} // a,b为0.05和0.25，或者0.15和0.15则可以通过，如果为0.1和0.2则无法通过。


数值范围：

  • ECMAScript能过表示的最小数值保存在Number.MIN_VALUE中——在大多数浏览器中，这个值是5e-324。
  • 能够表示的最大数值保存在Number.MAX_VALUE——在大多数浏览器中，这个值是1.7976931348623157e+308。
  • 如果超出范围则显示Infinity,负值则为-Infinity。使用isFinite()可以来测试某个数值是否处于最大最小之间。
